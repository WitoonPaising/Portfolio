ThiaoChumChon

คำอธิบาย

แอปส่งเสริมการท่องเที่ยวชุมชนที่ออกแบบมาเพื่อเชื่อมโยงนักท่องเที่ยวกับผู้ประกอบการและชุมชนท้องถิ่น ผู้ใช้สามารถเลือกกิจกรรมที่หลากหลาย เช่น ล่องแพ สำรวจถ้ำ ทัวร์เชิงนิเวศ หรือเวิร์กช็อปวัฒนธรรมท้องถิ่น พร้อมทั้งสามารถดูรายละเอียดกิจกรรม รีวิวจากผู้ใช้คนอื่น ๆ และจองกิจกรรมผ่านแอปได้อย่างสะดวก ระบบรองรับการจัดการจอง การแจ้งเตือนสถานะการจอง และมีฟีเจอร์แนะนำกิจกรรมตามความสนใจของผู้ใช้ ช่วยส่งเสริมเศรษฐกิจในชุมชนและสร้างประสบการณ์ท่องเที่ยวที่ยั่งยืนและมีคุณค่า

หน้าที่ที่รับผิดชอบ (Tester)

Manual Testing: ออกแบบ Test Case และดำเนินการทดสอบแอปพลิเคชันเพื่อค้นหา Bug และตรวจสอบความถูกต้องของฟังก์ชัน พร้อมออกแบบและบันทึกผลใน Google Sheet อย่างเป็นระบบ ทำงานร่วมกับ ทีมพัฒนา และ นักออกแบบ UX/UI เพื่อเสนอแนวทางแก้ไขปัญหาที่พบ และปรับปรุงประสบการณ์ผู้ใช้

Automated Testing: พัฒนาและรันสคริปต์ด้วย Robot Framework และ Appium Library เพื่อทดสอบฟังก์ชันซ้ำ ๆ บน Android Studio Emulator พร้อมบันทึกผลการทดสอบลง Google Sheet แบบอัตโนมัติและเรียลไทม์

Security Testing (Pentest เบื้องต้น): ตรวจสอบช่องโหว่ SQL Injection – Login Bypass และจัดทำรายงานผลการทดสอบเพื่อส่งมอบให้ทีมพัฒนา

Requirement Gathering: ประสานงานและรวบรวมความต้องการ (Requirements) จากผู้ใช้งานจริงในพื้นที่ เพื่อใช้ในการพัฒนาฟีเจอร์ให้ตรงกับความต้องการ
